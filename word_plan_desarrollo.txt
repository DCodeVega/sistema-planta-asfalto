🧭 ¿Cómo usar este plan?
No saltes de un hito a otro sin completar el anterior. Cada uno se basa en el previo.

Después de cada tarea, haz commit a Git con un mensaje claro (ej: git commit -m "feat: add material CRUD with unit").

Al final de cada hito, haz una prueba manual completa (o automatizada) antes de continuar.

Si encuentras un error o duda, escríbeme exactamente qué tarea y qué mensaje de error sale. Lo resolvemos.
